# TL40-Dots — Simple dotfiles for beginners (no backups)

This repository contains your personal configuration files ("dotfiles") organized into folders that can be linked into your home folder with GNU Stow.

Important: you asked for no backups. The commands below will permanently remove or overwrite existing files in your home directory. Only run them if you understand they will delete the current files.

## Configs in this repo:
- `aichat`
- `atuin`
- `burn-my-windows`
- `gsnap`
- `guake`
- `kitty`
- `starship`
- `.bash.rc`
- `BlendOS system.yaml`
